# Music Store SQL_Project
PostgreSQL project demonstrating practical SQL queries and insights documentation for business decision making
## Project Overview
This project analyze a music store database using PostgreSQL. The goal is to explore customer behavior, sales performance, and music trends by writing structured SQL queries on real-world data.
## Tools Used
- PostgreSQL
## SQL Concepts Applied
- SELECT queries
- JOIN operations
- GROUP BY and ORDER BY
- Subqueries
- Aggregate functions
## Business Questions Solved
- Identified the senior most employee
- Found countries with the highest number of invoices
- Analyzed top revenue generating cities
- Identified the highest spending customer
- Identified Rock music listeners
- Found top Rock artists
- Analyzed tracks onger than average duration
## KEY Insights
- High-value customers contribute significantly to total revenue
- Certain cities and countries genrate higher sales
- Rock genre shows strong customer engagement
- Some artists dominate in terms of track count
## Conclusion
This project demonstrates practical Sql skills used to solve real business problems. It highlights analytical thinking and database querying abilites.
